# CoalNext

A data engineering project focused on coal industry analytics — processing, transforming, and analyzing coal production, trade, and pricing data using modern data pipeline tooling.

## Overview

This repository covers end-to-end data workflows for coal industry datasets including extraction from public data sources, transformation pipelines, and analytics-ready output.

## Technology Stack

| Tool | Purpose |
|------|---------|
| **Python** | Core data processing |
| **pandas** | Data transformation |
| **SQL** | Analytics queries |
| **Jupyter Notebook** | Exploratory analysis |

## Project Structure

```
coal-next/
├── notebooks/          # Exploratory analysis notebooks
├── data/               # Raw and processed datasets (gitignored)
├── scripts/            # ETL and transformation scripts
├── .gitignore
└── README.md
```

## Quick Start

```bash
# Install dependencies
pip install pandas jupyter

# Launch notebooks
jupyter notebook
```

## Key Focus Areas

- Coal production and trade statistics
- Price trend analysis
- Regional demand forecasting
- Data pipeline automation
